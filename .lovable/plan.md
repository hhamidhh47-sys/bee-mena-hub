

## خطة التنفيذ

### 1. بيانات تجريبية (`src/lib/db.ts`)
- رفع إصدار قاعدة البيانات إلى `7`
- إضافة في `seedDatabase`:
  - **4 عملاء**: عبدالله العتيبي، سعود المطيري، فهد الشمري، خالد الحربي
  - **3 عناصر مخزون خلايا** (متاح/محجوز/مباع) + **3 عناصر عسل**
  - **3 فواتير**: مدفوعة بالكامل، مدفوعة جزئياً، غير مدفوعة
  - **دفعات** مرتبطة بالفواتير الجزئية والمدفوعة

### 2. تعديل الفواتير (`src/pages/StorePage.tsx` - `InvoiceViewDialog`)
- إضافة حالة `isEditing` مع زر "تعديل" بجوار عنوان الفاتورة
- في وضع التعديل:
  - تغيير العميل (Popover + Command للبحث)
  - إضافة/حذف منتجات من القائمة
  - تعديل الملاحظات وتاريخ الاستحقاق
- زر "حفظ" يستدعي `updateInvoice` مع إعادة حساب الحالة
- زر "إلغاء" يعود لوضع العرض

### 3. صفحة تفاصيل العميل (`src/pages/StorePage.tsx` - `CustomersTab`)
- عند الضغط على عميل تظهر نافذة `Dialog` تعرض:
  - بيانات العميل الأساسية (الاسم، الهاتف، الموقع)
  - إحصائيات: إجمالي الفواتير، إجمالي المدفوع، إجمالي الديون
  - قائمة فواتيره مع حالتها (بادج ملونة) وإمكانية فتح أي فاتورة
- استخدام `useInvoices(customerId)` لجلب فواتير العميل

### الملفات المتأثرة
| الملف | التغيير |
|---|---|
| `src/lib/db.ts` | رفع الإصدار + بيانات تجريبية |
| `src/pages/StorePage.tsx` | تعديل الفواتير + تفاصيل العميل |
| `src/hooks/useDatabase.ts` | لا تغييرات (الدوال موجودة) |
